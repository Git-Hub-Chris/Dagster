# Welcome to Dagster!

<p className="text-2xl mt-0 text-gray-500 tracking-tight font-light">
  Dagster is the data orchestration platform built for productivity.
</p>

Dagster is an orchestrator designed for developing and maintaining data assets, such as tables, data sets, machine learning models, and reports.
