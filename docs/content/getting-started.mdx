## Installing Dagster

To install Dagster into an existing Python environment, run:

```bash
pip install dagster
```

This will install the latest stable version of the core Dagster packages in your current Python environment.

---

## Writing a job

To get started, we'll write some simple [ops](/concepts/ops-jobs-graphs/ops) and use them to create a [job](/concepts/ops-jobs-graphs/jobs-graphs). Dagster also offers a way to produce data artifacts like tables and files using [software-defined assets](/concepts/assets/software-defined-assets), but for simplicity we'll focus on ops.

Copy the following code and save it in a file named `hello_world.py`:

```python file=/getting_started/hello_world.py startafter=start_pipeline_marker endbefore=end_pipeline_marker
from dagster import job, op


@op
def get_name():
    return "dagster"


@op
def hello(name: str):
    print(f"Hello, {name}!")


@job
def hello_dagster():
    hello(get_name())
```

## Running the job

In Dagster, jobs can be executed using the:

- [Dagit](#using-dagit)
- [Dagster Python API](#using-the-dagster-python-api)
- [Dagster CLI](#using-the-dagster-cli)

### Using Dagit

1. [Dagit](/concepts/dagit/dagit) is a web-based interface for viewing and interacting with Dagster objects. To install Dagit, run:

   ```bash
   pip install dagit
   ```

1. To visualize your job in Dagit:

   ```bash
   dagit -f hello_world.py
   ```

1. Navigate to <http://localhost:3000> to start using Dagit.
1. Click the **Launchpad** tab:

   <Image
   alt="dagit-def"
   src="/images/getting-started/dagit-def.png"
   width={4032}
   height={2454}
   />

1. Lastly, click the **Launch Run** button in the bottom right corner to launch the job:

   <Image
   alt="dagit-run"
   src="/images/getting-started/dagit-run.png"
   width={4032}
   height={2454}
   />

### Using the Dagster Python API

You can also use the [Dagster Python API](/concepts/ops-jobs-graphs/job-execution#python-apis) to execute jobs:

```python file=/getting_started/hello_world.py startafter=start_execute_marker endbefore=end_execute_marker
if __name__ == "__main__":
    result = hello_dagster.execute_in_process()
```

### Using the Dagster CLI

Or the [Dagster CLI](/\_apidocs/cli#dagster-pipeline-execute):

```bash
dagster job execute -f hello_world.py
```
